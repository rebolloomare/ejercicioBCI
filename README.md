## Información General
El objetivo de este proyecto es desarrollar un microservicio para la creación y consulta de usuarios.

## Tecnologías
El proyecto ha sido creado con las siguientes herramientas y versiones:
* Java 11
* Spring Framework 5.3.20
* SpringBoot 2.5.14
* Gradle 7.4
* Lombook 1.8.30
* JUnit 5
